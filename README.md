# Flair

## The tool I wish I had to inform my learning

In Scott Young's book Ultralearning, the first step is 'metalearning' (learning how to learn). This is why I have built _Flair_; to give you a roadmap with resources for metalearning.

## Contributing

If you would like to contribute, add or update a respository/index.json with the relevant resources. Create a PR. Providing its reasonable, I will accept. Self promotion is fine, but your resource has to be useful and **easy to use.**
